# Aswang Hunter
An Augmented Reality Zombie Hunting Game with 4 different levels.It includes:<br>
- A kill timer <br>
- A round timer <br>
- An animated virtual sword  you use to kill zombies <br>
- Background music <br>
- Sound fx

# Skills
- Swift 3 -> Swift 4
- ARKit
- Haptics
- AVAudioKit
- AutoLayout
- MVC
- Design / UI
