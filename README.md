# Aswang Hunter
An Augmented Reality Zombie Hunting Game with 4 different levels.It includes:<br>
a kill timer <br>
a round timer <br>
an animated virtual sword  you use to kill zombies <br>
background music <br>

# Skills
- Swift 3 -> Swift 4
- ARKit
- Haptics
- AVAudioKit
